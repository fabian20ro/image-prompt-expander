How it works: A local LLM (GLM-4.7-Flash via LM Studio) analyzes your input prompt and generates a Tracery grammar that preserves specified elements (dragon, flying) while creating variations for unspecified elements (dragon type, environment, lighting, etc.). Tracery then expands this grammar into unique prompt combinations.
